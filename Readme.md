intalamos python y usamos
    > pip install virtualenv
luego iniciamos el entorno virtual
    > virtualenv (NombreProyecto)
luego activamos el entorno
    > .\venv\Scripts\activate
instalamos Django
    > pip install django
creamos un proyecto Django
    > django-admin startproject (NombreCarpeta) .
creamos sub aplicaciones
    > python manage.py startapp (NombreCarpetaSubApp)
lanzamos la app 
    > python manage.py runserver (Puerto, es opcional)
